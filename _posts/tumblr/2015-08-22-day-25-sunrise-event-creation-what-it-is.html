What it is: Sunrise is a calendar app that acts as an intermediary between all of your other calendars. I started using it because it would pull my due dates from Trello, but I’ve also found that it integrates with just about every calendar service.

This is how you create an event in Sunrise, and some of the options available to you.

Why it’s good: Two wordsâ€”consolidation and suggestion. First, every conceivable option that you can apply to this event is rolled into one densely packed interface. It is dense, but the flow makes complete sense because it feels like a list of questions, and that list matches how I think about my events:

  • What am I doing?
  • When?
  • Where?
  • Who?
  • Anything Else?

It rolls in smart suggestions, like having an incredibly fast and accurate calendar lookup, pulling in my address book for invitations, and even subtle things like changing the icon based on what the event is called.

My favorite piece of it though is that Sunrise allows you to create an event without asking you which calendar it’s on. This is incredibly important to an aggregation tool like this - I don’t want to think about how I have to create the event, I just want to create it. While the interface could be better, making the calendar the last choice before you save instead of the first choice (Like Outlook for instance) removes so much mental friction from the process, and allows me to focus on the more relevant details.
