What it is: When listening to Spotify’s radio, the thumbs up icon tells you that it’s improving the station before adding it to a playlist for you.

Why it’s good: First, the pause is brilliant. I’m sure Spotify doesn’t need any processing time to pick the next song, but this brief pause makes me feel like they’re actually doing something important. The reason that it works to make me wait for a second is that it’s unlikely at this point that I would want to do something else, so the action doesn’t block me.

The part that elevates this to an amazing design is automatically adding it to a playlist. Liking a song to improve a station is a concept that I’m used to because of Pandora. Adding it to a list where I can find it later (and play it by itself) is a direct sell of Spotify’s added value - You can listen to it on the radio, but you can also listen to it by itself. Even though it’s subtle, this interface piece underscores that value proposition.
